HABDORN PDF - PRIMERA VERSION
==============================

FUNCIONES
- Abrir uno o varios archivos PDF.
- Añadir imágenes como páginas A4.
- Añadir páginas A4 en blanco.
- Reordenar páginas arrastrando sus miniaturas.
- Eliminar páginas.
- Insertar imágenes sobre una página existente.
- Mover y redimensionar las imágenes insertadas.
- Exportar todo a un PDF nuevo.

EJECUCIÓN RÁPIDA
1. Instala Python 3.11 o 3.12 desde python.org.
2. Durante la instalación, marca "Add Python to PATH".
3. Haz doble clic en EJECUTAR_EN_WINDOWS.bat.

CREAR EL EJECUTABLE
1. Haz doble clic en CREAR_EXE_WINDOWS.bat.
2. Al terminar, abre:
   dist\HabdornPDF\HabdornPDF.exe
3. Para llevarlo a otra computadora, copia toda la carpeta:
   dist\HabdornPDF

USO
- Usa "+ PDF" para cargar documentos.
- Usa "+ Imagen como página" para convertir una imagen en una página A4.
- Arrastra las miniaturas para cambiar el orden.
- Selecciona una página y pulsa "Insertar imagen".
- Arrastra la imagen para moverla.
- Selecciónala y arrastra el cuadro azul inferior derecho para cambiar su tamaño.
- Pulsa Supr para eliminar una imagen seleccionada.
- Pulsa "Exportar PDF" para guardar el resultado.

NOTAS
- El programa trabaja localmente y no sube archivos a internet.
- Los PDF protegidos con contraseña todavía no están soportados.
- Esta versión no edita texto existente.
